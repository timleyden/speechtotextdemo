Show Account Details: This button toggles the account details form.

This form collects the information required to authenticate with the transcription service as well as the storage account hosting the audio files. To use transcriptionViewer you will need to provide the following details

AccountName: The name of the storage account used to store the audio files (without blob.core.windows.net) More Details

SASToken: This is the sas token used to authenticate with the blob serice or the container used to store the audio files. It should start with a '?'.More Details

Speech Region: you need to specify the region you created you Speech services subscription in. We need this to connect to the appropriate endpointMore Details

Service Key: you need to specify your speech to text service key. this is effectivly an api key used to authenticate your request against your subscription More Details
